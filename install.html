Install instructions

  1. COPY the folder "The Sims 2 Ultimate Collection" to the desired install location (program files).
  2. Inside the subfolder "__Installer," Right-click on the file "Touchup.exe" and create a shortcut. It is possible that you may get an error message that the shortcut "cannot be created in this location." If so, send it to the desktop.
  3. Right-click on the shortcut and select "Properties," then at the end of the "Target" line, paste the following without brackets, including the space at the beginning: [ install -locale en_US -installPath "C:\Program files\The Sims 2 Ultimate Collection" -autologging]. When you're done, the line will say ""C:\Program Files\The Sims 2 Ultimate Collection\__Installer\Touchup.exe" install -locale en_US -installPath "C:\Program files\The Sims 2 Ultimate Collection" -autologging".
  4. Double-click on the "Touchup.exe - Shortcut", and ... wait for about 10 seconds. Touchup.exe will install the game silently in the background with the paramaters that were entered into the Target.
  5. Copy the Sims2EP9.exe from the CRACK folder to Fun With Pets/SP9/TSBin and overwrite the original file.

Once the game is installed, move on to the Guide for modern systems.

Further troubleshooting can be found in the installation guide that comes with the torrent.

DO NOT use the Graphics Rules Maker that comes with the torrent, it is old and doesn't have the new fixes needed to run the game.
